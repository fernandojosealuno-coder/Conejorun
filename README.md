# Conejo Run

Jogo desenvolvido em Java Swing.

## Como jogar

- Pressione **Espaço** ou **seta para cima** para pular.
- Desvie dos troncos.
- Colete cenouras para ganhar pontos extras.
- Pressione **Espaço** após perder para reiniciar.

## Executar

```bash
javac src/App.java src/ConejoGame.java
java -cp src App
```
